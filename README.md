# LBP1

Configuração de ambiente virtual e instalação do Flask para desenvolvimento do projeto.

## Instruções

### Ambiente Virtual(Python)

Crie o ambiente virtual usando o comando abaixo:

```bash
python3 -m venv AMB
source AMB/bin/activate
pip install flask
```
### Windows
```
AMB\Scripts\activate
```
